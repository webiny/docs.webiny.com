## Grid Block

Grid block is the fundamental building block on a page. When you create a page, the first thing to add is a grid block.
It serves as the parent container for all the page elements.

## Page Element

Page Builder application comes with a plethora of ready-made page elements like Heading, Paragraph, Quote, Image, etc.
These are the building blocks of a page.
