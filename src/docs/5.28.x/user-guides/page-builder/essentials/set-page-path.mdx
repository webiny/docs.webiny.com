In this tutorial, we will learn how to set the path used to access a page. This will be demonstrated using 
the **Welcome to Webiny** example project.

<Video src={createContent} controls={true}/>

## Set Page Path
1. From the home page, click on **BUILD A NEW PAGE** in the Page Builder card.  
2. Select the page you want to set the path. For this tutorial, we will select **Welcome to Webiny**.
3. In the page preview, click on the edit icon.

   ✔️ The page editor opens.
4. Click on the settings icon at the top-right of the screen.
5. In the **Path** textbox, set your desired path.
6. Click **SAVE SETTINGS** at the bottom of the page.

   ✔️ The page editor opens.
7. Click **PUBLISHE CHANGES** at the top-right of the page.

   ✔️ A modal opens.
8. Click **CONFIRM**.
