## `AdminStackOutputService`

**Abstraction** — imported from `webiny/extensions`

```typescript
import { AdminStackOutputService } from "webiny/extensions";
```

**Interface `AdminStackOutputService.Interface`:**

```typescript
interface AdminStackOutputService.Interface {
    execute<TOutput extends IAdminStackOutput = IAdminStackOutput>(): Promise<TOutput | null>;
}
```

**Types:**

```typescript
namespace AdminStackOutputService {
  type Interface = IAdminStackOutputService;
  type Output = IAdminStackOutput;
}
```

**Usage:**

```typescript
// extensions/MyImpl.ts
import { AdminStackOutputService } from "webiny/extensions";

class MyImpl implements MyUseCase.Interface {
  public constructor(private adminStackOutputService: AdminStackOutputService.Interface) {}

  public async execute(/* ... */): Promise<void> {
    await this.adminStackOutputService.execute(/* ... */);
  }
}

export default MyUseCase.createImplementation({
  implementation: MyImpl,
  dependencies: [AdminStackOutputService]
});
```
