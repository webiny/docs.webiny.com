## Overview

The `webiny/extensions` package exports React components used inside `webiny.config.tsx` to wire extensions into your Webiny project. Each component corresponds to a `defineExtension()` call in the Webiny source and accepts typed props defined by its Zod schema.

## EnvVar

### `EnvVar`

Set an environment variable in the project context.

Can be used **multiple times**.

**Props**

| Prop      | Type     | Required | Description                     |
| --------- | -------- | -------- | ------------------------------- |
| `varName` | `string` | yes      | The environment variable name.  |
| `value`   | `string` | yes      | The environment variable value. |

```tsx webiny.config.tsx
import { EnvVar } from "webiny/extensions";

export const Extensions = () => <EnvVar varName="value" value="value" />;
```

## FeatureFlags

### `FeatureFlags`

Enable or disable WCP features.

Can only be used **once**.

**Props**

| Prop                           | Type      | Required | Description |
| ------------------------------ | --------- | -------- | ----------- |
| `features`                     | `object`  | no       | —           |
| ↳ `multiTenancy`               | `boolean` | no       | —           |
| ↳ `advancedPublishingWorkflow` | `boolean` | no       | —           |
| ↳ `advancedAccessControlLayer` | `union`   | no       | —           |
| ↳ `auditLogs`                  | `boolean` | no       | —           |
| ↳ `recordLocking`              | `boolean` | no       | —           |
| ↳ `fileManager`                | `object`  | no       | —           |

```tsx webiny.config.tsx
import { FeatureFlags } from "webiny/extensions";

export const Extensions = () => <FeatureFlags />;
```

### `Vpc`

Can only be used **once**.

**Props**

| Prop              | Type      | Required | Description                                       |
| ----------------- | --------- | -------- | ------------------------------------------------- |

| `useVpcEndpoints` | `boolean` | no       | Whether to use VPC endpoints for AWS services.    |

import { Infra } from "webiny/extensions";

export const Extensions = () => <Infra.Vpc />;
