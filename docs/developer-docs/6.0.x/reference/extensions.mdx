## Overview

This page documents everything exported from `webiny/extensions`. Import any of the items below directly from this path in your Webiny extensions.

## `EnvVar`

**Constant** — imported from `webiny/extensions`

```typescript
import { EnvVar } from "webiny/extensions";
```

```typescript
export const EnvVar = defineExtension({
  type: "Project/EnvVar",
  tags: { runtimeContext: "project" },
  description: "Set an environment variable in the project context.",
  multiple: true,
  paramsSchema: z.object({
    // TODO: enable using `name` instead of `varName` for better consistency.
    varName: z.string().describe("The environment variable name."),
    value: z.string().describe("The environment variable value.")
  })
});
```
