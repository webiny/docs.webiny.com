- Applications communicate via GraphQL and events

Core Applications:
1. Headless CMS - Content management and API
2. Website Builder - Visual page creation
3. File Manager - Digital asset management
4. Publishing Workflows - Content approval pipeline
5. Admin - Unified control panel

Architecture Details:
- Deploy across Core/API/Admin stacks
- Share infrastructure (Cognito, S3, etc.)
- Communicate via GraphQL and EventBridge
- Each app provides extension points

Key Integration Points:
- CMS stores data for other apps
- File Manager provides asset selection UI
- Admin hosts all application UIs
- Events allow cross-app communication
