WCP Understanding:
- WCP is only needed for paid features, not for Community edition
- WCP projects are separate from Webiny instances
- One WCP project can have multiple linked environments
- Team members in WCP are different from users in Webiny instance
- License activation requires redeployment

Key Commands:
- `yarn webiny link-project --project-id=<ID>` - Link instance to WCP project
- `yarn webiny link-project --env=<env> --project-id=<ID>` - Link specific environment
- `yarn webiny deploy` - Required after linking to activate license
