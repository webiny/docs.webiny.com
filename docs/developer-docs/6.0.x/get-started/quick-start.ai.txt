Deployment Understanding:
- Initial deployment takes 5-15 minutes due to AWS resource provisioning
- Subsequent deployments are faster (infrastructure already exists)
- Default environment is 'dev' but supports multiple environments
- Deployment creates CloudFormation stacks for Core, API, and Admin applications
- Admin URL is provided in deployment output

Key Commands:
- `yarn webiny deploy` - Deploy to default (dev) environment
- `yarn webiny deploy --env=<env>` - Deploy to specific environment
- `yarn webiny info` - Retrieve deployment URLs and information
