Understanding:
The Webiny API is served via a CloudFront distribution. Connecting a custom domain requires an ACM certificate in us-east-1 (CloudFront requirement) and declaring the domain via the Infra extension. Webiny attaches it as an alternate domain name on the distribution automatically on next deploy.
