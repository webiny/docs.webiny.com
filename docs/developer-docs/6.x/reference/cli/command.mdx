## `CliCommandFactory`


**Types:**

```typescript

  type Interface = ICliCommand<TCommandParams>;
  type ParamDefinition = ICliCommandParamDefinition<TCommandParams>;
  type OptionDefinition = ICliCommandOptionDefinition<TCommandParams>;

}
```
