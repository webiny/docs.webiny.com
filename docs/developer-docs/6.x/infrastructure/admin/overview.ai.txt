Understanding Admin Area Architecture:
- The Admin Area is a standard React SPA built with create-react-app (or similar)
- During deployment, the React app is built (yarn build) and uploaded to S3
- S3 is configured for static website hosting with index.html as the default document
- CloudFront points to the S3 bucket as its origin
- All client-side routing is handled by React Router (or similar)
- CloudFront is configured to return index.html for all routes (SPA routing pattern)
- API requests from the Admin Area go to a completely separate CloudFront distribution (the API infrastructure)

SPA Hosting Pattern:
- S3 stores the files (HTML, JS, CSS, images, fonts)
- CloudFront distributes globally and caches at edge locations
- Browser loads index.html, which loads JavaScript bundles
- JavaScript handles routing, UI, and API calls
- This is distinct from SSR or static site generation
