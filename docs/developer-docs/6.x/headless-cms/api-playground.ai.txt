Key Facts:
- Playground is accessed via Support icon (bottom of left sidebar) in Webiny Admin
- Four tabs: Main API, Headless CMS Manage API, Read API, Preview API
- Auto-authenticated as the logged-in Admin user — no manual headers needed inside Admin
- Queries are NOT persisted across page refreshes
- Auto-complete: Ctrl+Space / Cmd+Space
- HTTP Headers tab is how external clients add API key auth
