## Introduction

It's not uncommon to have a page element that needs to load data from an external source. For example, you might want to display a list of products, a list of blog posts, or a list of categories in a Page Builder element. In this guide, you'll learn how to efficiently load data in Page Builder elements using the `useLoader` React hook.
