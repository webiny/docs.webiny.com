Webiny 6.4.4 is out! :rocket:

This release brings a nice quality-of-life improvement for teams using external identity providers — you can now set up federated sign-in through Amazon Cognito, letting users authenticate via Google, Microsoft Entra ID, or any OIDC-compliant provider. No more managing separate Webiny credentials when your org already has an IdP in place.

We've also squashed a few bugs worth mentioning: scheduled publishing and unpublishing of CMS entries now works properly, and custom production environments are correctly recognized during deployment. There's also an important security fix in the Headless CMS, so upgrading sooner rather than later is a good idea. Check out the changelog to learn more.
