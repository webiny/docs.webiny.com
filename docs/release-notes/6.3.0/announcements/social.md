Webiny 6.3.0 is out.

This release starts rolling out AI PowerUps — initially for paid plans. Connect OpenAI or Anthropic in Admin settings to generate page content directly in Website Builder using natural language prompts. Images uploaded to File Manager are also now automatically enriched with AI-generated tags and descriptions.

For developers, there's a new declarative FormModel API for building admin forms with tabs, validation, computed fields, and dynamic zones. The SDK gains a tasks module for triggering and managing background tasks from external apps, and all SDK methods now validate inputs before hitting the network.

Check the changelog and let us know what you think.
