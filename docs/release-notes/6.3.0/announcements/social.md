Webiny 6.3.0 is here 🚀

This release introduces AI PowerUps — a new feature set that brings AI-powered content generation to the Website Builder and automatic image enrichment to the File Manager. Configure your OpenAI or Anthropic provider in Admin settings, then generate page sections using natural language prompts or let uploaded images automatically receive AI-generated tags and descriptions.

For developers, there's a new declarative FormModel API for building admin forms, a Tasks SDK module for interacting with background tasks from external applications, and improved input validation across all SDK methods. The encryption service is now built in, TypeScript has been upgraded to 6.0.3, and Pulumi extensions finally have proper typing.

Infrastructure improvements include self-cleaning background tasks, mailer configuration via webiny.config.ts, and fixes for duplicate resource URNs when registering multiple API routes.

We'd love to hear what you build with AI PowerUps — share your experience!
