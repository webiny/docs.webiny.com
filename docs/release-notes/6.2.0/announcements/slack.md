Webiny 6.2.0 is out! :rocket:

This release brings some exciting multilingual capabilities to Page Builder! You can now create pages in specific languages and translate existing pages directly from the pages list. The editor also shows language indicators so you always know which version you're working on.

We've also made nice improvements to the SDK and developer experience. The new `sdk.languages` module makes it easy to work with configured languages, and the token parameter now supports async functions for dynamic token refresh. Plus, there's a new `Api.Route` extension that lets you register custom REST routes with full dependency injection support.

On the infrastructure side, you can now configure custom domains for your API CloudFront distribution directly in your config file. Check out the changelog to learn more.
