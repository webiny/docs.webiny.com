## Step-by-Step Guide

### 1. Upgrade Webiny Packages

Upgrade all Webiny packages by running the following command:

```bash
yarn webiny upgrade 6.2.0
```

Note that the command above will run upgrades for all available versions of Webiny up to 6.2.0. If there are upgrades for 6.1.1, 6.1.5, they will be ran.

You can omit the version to upgrade to the latest available:

```bash
yarn webiny upgrade
```

Once the upgrade has finished, running the `yarn webiny --version` command in your terminal should return **6.2.0**.

### 2. Deploy Your Project

Proceed by redeploying your Webiny project:

```bash
# Execute in your project root.
yarn webiny deploy --env {environment}
```
