Webiny 6.4.2 is out! :rocket:

This is a small but solid patch release focused on fixing a few frustrating bugs. If you've ever tried to re-publish a CMS entry that was previously unpublished, you'll be happy to know that's now working as expected.

We also tackled a browser compatibility issue with date formatting in the Admin UI, and tidied up some visual quirks in the Website Builder page editor — input fields now look consistent and that jumpy orange selection border is finally staying put. Check out the changelog to learn more.
