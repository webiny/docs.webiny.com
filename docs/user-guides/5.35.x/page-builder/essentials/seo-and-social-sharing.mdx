In this tutorial, we will learn how to edit SEO and social sharing settings of a page. We will do this in 2 steps:

- Step 1: Edit social sharing settings
- Step 2: Edit SEO settings

## Prerequisites
To follow this tutorial you need the **The benefits of a Serverless CMS** page. If you haven't created this page, please follow the [Create Page](/docs/{version}/user-guides/page-builder/essentials/create-page) tutorial to create it.

## Step 1: Edit Social Sharing Settings

1. From the **Side Menu**, click **Page Builder** > **PAGES** > **Pages**.

    ✔️ The **Pages** screen opens.

2. Click the **The benefits of a Serverless CMS** list item in the list of pages.

    ✔️ The **PAGE PREVIEW** tab opens.

3. In the **PAGE PREVIEW** tab, click the **EDIT** icon (✏️).

    ✔️ The page editor screen opens.

4. Click the **SETTINGS** icon (⚙️).

    ✔️ The Page Settings screen opens.

5. Click the **Social media** tab.

    ✔️ The **Social Media** settings screen opens.

6. In the **Title** textbox, type **What are the benefits of a Serverless CMS**.

    **Note**: This text will be displayed as the page's title when its link is shared over social media.

7. In the **Description** textbox, type **Increases business resiliency | Reduces infrastructure spend**. 
    
    **Note**: This text will be displayed as the page's description when its link is shared over social media.

8. In the **Social Image** image selector, select the image *you want to be displayed as the page's thumbnail* when its link is shared over social media.

9. Click **SAVE SETTINGS**.

    ✔️ The page editor screen opens with the message "Settings saved!". 


## Step 2: Edit SEO Settings

1. Click the **SETTINGS** icon (⚙️).

    ✔️ The **Page Settings** screen opens.

2. Click the **SEO** tab.

    ✔️ The **SEO Media** settings screen opens.

3. In the **Title** textbox, type **The benefits of a Serverless CMS**.

4. In the **Description** textbox, type **Increases business resiliency | Reduces infrastructure spend**.

**Note**: The SEO Title and SEO Description correspond to the Title tag and Meta Description tag of HTML respectively. The purpose of these tags is to inform search engines like Google about the information contained in your web page. The SEO Title and SEO Description of a page shows up on the search results page of a browser when user makes a search query.

5. Click **SAVE SETTINGS**.

    ✔️ The page editor screen opens with the message "Settings saved!".
